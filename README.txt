Informations                
==================================================================================

Le rendu est divisé en plusieurs fichiers : phrase_reflexive.ml , phrase_reflexive.mli et tests.ml 
Ces fichiers et le Makefile sont situés dans le dossier src. Le rapport est dans doc.

La commande suivante permet de compiler le programme et lancer les tests unitaires :
> cd src && make && ./test && cd ..

Toutes les fonctions testables sont testées. À la fin du fichier tests.ml, le générateur est lancé sur peu de lettres pour obtenir un résultat rapidement. Vous pouvez le lancer sur plus de lettres, jusqu'à 26 lettres pour la méthode d'Hofstadter sans problème. Pour la méthode de Pitrat, au delà de 20 lettres, le programme prend vraiment beaucoup de temps. Des tests gourmands en ressources sont exposés dans le rapport.
